## 📌 프로젝트 소개

**DevView**는 취업 준비생과 이직 준비자들이 언제 어디서나 실전과 같은 면접을 연습할 수 있도록 돕는 AI 기반 면접 준비 서비스입니다. 

### 🎯 목표
- **접근성**: 시간과 장소에 구애받지 않은 면접 연습
- **실전감**: AI를 활용한 실제 면접과 유사한 경험 제공
- **맞춤형**: 직무와 경력 수준에 맞는 개인화된 질문
- **성장**: 상세한 피드백과 학습 자료 제공

### 🤔 해결하고자 하는 문제
- 면접 경험 부족으로 인한 실전 대응력 저하
- 시간과 비용 제약으로 인한 면접 스터디 참여 어려움
- 객관적인 피드백 부재로 인한 개선점 파악 어려움

## ✨ 주요 기능

### 1. 🤖 AI 면접 진행
- **맞춤형 질문 생성**: Gemini API를 활용한 직무/경력별 질문
- **4가지 면접 유형**: 기술(Technical), 실무(Practical), 인성(Behavioral), 종합(Comprehensive)
- **실시간 답변 분석**: AI가 답변을 분석하여 즉시 피드백 제공

### 2. 📊 면접 결과 분석
- **종합 점수 산정**: 기술력, 문제해결, 소통능력, 태도 등 다각도 평가
- **상세 피드백**: 강점과 개선점을 구체적으로 제시
- **등급 시스템**: S, A, B, C, D 5단계 등급으로 실력 파악

### 3. 📚 학습 자료 추천
- **Alan API 연동**: 부족한 부분에 대한 학습 자료 자동 검색
- **키워드 기반 추천**: 면접 내용 분석 후 핵심 키워드 추출
- **맞춤형 리소스**: 개인별 약점 보완을 위한 자료 제공

### 4. 💬 커뮤니티
- **면접 후기 공유**: 실제 면접 경험 공유 게시판
- **좋아요/스크랩**: 유용한 게시글 저장 기능
- **경험 교류**: 다른 사용자들과 팁 공유

### 5. 👤 마이페이지
- **프로필 관리**: 직무, 경력 수준 설정
- **면접 이력**: 과거 면접 기록 및 성장 추이 확인
- **프로필 이미지**: 이미지 업로드/수정/삭제

### 6. 🔐 소셜 로그인
- **Google OAuth2**: 간편한 구글 계정 로그인
- **보안 강화**: Spring Security 기반 안전한 인증

## 🛠 기술 스택

### Backend
- **Framework**: Spring Boot 3.5.4
- **Language**: Java 17
- **Security**: Spring Security + OAuth2
- **Database**: PostgreSQL
- **ORM**: Spring Data JPA

### Frontend
- **Template Engine**: Thymeleaf
- **Styling**: CSS3
- **Script**: JavaScript (Vanilla)
- **UI Components**: Custom Components

### External APIs
- **Gemini API**: 면접 질문 생성 및 답변 분석
- **Alan API**: 학습 자료 검색 및 추천

### Infrastructure
- **Build Tool**: Maven
- **Container**: Docker Compose
- **Deployment**: AWS (예정)

## 🚀 설치 및 실행

### 필수 요구사항
- Java 17 이상
- Maven 3.6 이상
- PostgreSQL 14 이상
- Docker & Docker Compose (선택사항)

### 환경 변수 설정
`.env` 파일을 프로젝트 루트에 생성하고 다음 내용을 추가:

```env
# Database
DEV_DB_PASSWORD=your_db_password

# Google OAuth2
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret

# AI APIs
GEMINI_API_KEY=your_gemini_api_key
ALAN_CLIENT_ID=your_alan_client_id
```

### 실행 방법

#### 1. 프로젝트 클론
```bash
git clone https://github.com/Jaykim98z/DevView.git
cd DevView
```

#### 2. 데이터베이스 설정
```bash
# PostgreSQL 실행 (Docker 사용 시)
docker-compose up -d postgres

# 또는 로컬 PostgreSQL 사용 시
createdb devview_dev
```

#### 3. 애플리케이션 실행
```bash
# Maven으로 실행
mvn spring-boot:run

# 또는 JAR 빌드 후 실행
mvn clean package
java -jar target/DevView-0.0.1-SNAPSHOT.jar
```

#### 4. 접속
브라우저에서 `http://localhost:8080` 접속

## 📊 데이터베이스 스키마

### 주요 테이블
- **users**: 사용자 정보
- **interviews**: 면접 세션
- **interview_questions**: 면접 질문
- **interview_answers**: 사용자 답변
- **interview_results**: 면접 결과 및 피드백
- **community_posts**: 커뮤니티 게시글
- **user_profiles**: 사용자 프로필 확장 정보

## 🔄 API 엔드포인트

### 면접 관련
- `POST /api/v1/interviews/start` - 면접 시작
- `GET /api/v1/interviews/{id}/questions` - 질문 조회
- `POST /api/v1/interviews/{id}/answers` - 답변 제출
- `POST /api/v1/interviews/{id}/end` - 면접 종료
- `GET /api/v1/interviews/{id}/result` - 결과 조회

### 커뮤니티
- `GET /api/community/posts` - 게시글 목록
- `POST /api/community/posts` - 게시글 작성
- `POST /api/community/posts/{id}/like` - 좋아요
- `POST /api/community/posts/{id}/scrap` - 스크랩

### 마이페이지
- `GET /api/mypage/profile` - 프로필 조회
- `POST /api/mypage/profile` - 프로필 수정
- `DELETE /api/mypage/profile/image` - 프로필 이미지 삭제
